About this Project

Cyclist-Fatalities.com aims to be a single location for information on all incidents of cyclists killed in vehicular crashes. All information is sourced from news publications. Data points such as coordinates of the crash are determined from information from these articles.

The goal of this project is to be a resource for cyclists, community activists, and researchers with an interest in eliminating cyclist deaths.
